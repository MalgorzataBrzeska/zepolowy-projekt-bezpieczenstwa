# zepolowy-projekt-bezpieczenstwa

Cel projektu:
Projekt polega na analizie bezpieczeństwa za pomocą webowej aplikacji. Umożliwa ona wykonywanie skanów systemu, sieci oraz plików lokalnych. Celem projektu jest zidentyfikowanie podatności bezpieczeństa,
ocena ryzyka oraz wdrożenie poprawek zgodnych z najlepszymi praktykami cyberbezpieczeństwa. 
Projekt realizowany był w formie zespołowej, z wykorzystaniem repozytorium Github.
